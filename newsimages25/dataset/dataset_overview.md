# NewsImages Dataset

This folder contains:

* the complete overview of all articles "news_articles.csv"
* the overview of all randomly picked articles for subtask 1 "subtask_1.csv" (also known as "SMALL")
* the overview of all randomly picked articles for subtask 2 "subtask_2.csv" (also known as "LARGE")

We created a separate archive where you can download all the original pictures.
It is available on the [UZH file server](https://seafile.ifi.uzh.ch/d/1f14d6d4306340e082d6) (see "newsimages_25_v1.2.zip").
